Week #1 Questions and Answers

HTML Question

Question: What is in the head?

Answer: The head of an HTML document is the part that is not displayed in the web browser when the page is loaded. It contains information such as the page title, links to CSS, links to custom favicons, and metadata (data about the HTML, such as who wrote it, and important keywords that describe the document)

CSS Question

Question: What is the selector in CSS?

Answer: Selector is the HTML element name at the start of the rule set. It selects the elements to be styled.

Website Structure Question 1

Question: What tags HTML provides? List five or more

Answer: Header, nav, main, section, aside, and footer

Website Structure Question 2

Question: What does the tag main use for?

Answer: The main tag specifies the main content of a document. The content inside the main element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, copyright information, site logos, and search forms
